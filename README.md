# asignads-core

Asigna es un SaaS que actua como ERP/CRM para empresas pymes del sector asegurador Colombiano.

## Stack Tecnologico

- **Backend**: Go
- **Base de Datos**: PostgreSQL
- **ORM**: GORM
- **Autenticación**: JWT
- **Testing**: Go Testing
- **Despliegue**: Docker
